# MCP Terminal

[iterm-mcp](https://github.com/ferrislucas/iterm-mcp)을 이용하면 mcp로 terminal을 실행할 수 있습니다. 이때, mcp config는 아래와 같이 설정합니다.

```java
{
  "mcpServers": {
    "iterm-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "iterm-mcp"
      ]
    }
  }
}
```
